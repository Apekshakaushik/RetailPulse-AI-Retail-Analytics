# 📊 RetailPulse AI – Retail Analytics Platform

An end-to-end AI-powered Retail Analytics project that transforms raw retail transaction data into actionable business insights using Data Analytics and Machine Learning.


## 🚀 Project Overview

RetailPulse AI helps businesses understand customer behavior, forecast future sales, identify customers at risk of churn, and optimize inventory planning.

The project follows a complete Data Science workflow from data preprocessing to predictive analytics and dashboard visualization.


## ✨ Features

- 📦 Data Cleaning & Preprocessing
- 📊 Exploratory Data Analysis (EDA)
- 👥 Customer Segmentation using RFM + K-Means Clustering
- 📈 Sales Demand Forecasting using Prophet
- ⚠️ Customer Churn Prediction using Machine Learning
- 📦 Inventory Optimization
- 🌐 Interactive Streamlit Dashboard
